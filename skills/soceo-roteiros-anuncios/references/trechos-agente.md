## Trecho: Diferenciar Roteiro De Instagram Ou Anuncio

> ANÁLISE: Identifique conceitos-chave e intenção da mensagem (caso seja um roteiro, pergunte se é um roteiro para INSTAGRAM ou ANÚNCIO )

Uso nesta skill: se o usuário pedir "roteiro" sem formato, perguntar se é anúncio, Reels, YouTube/VSL ou outro formato.

## Trecho: Formatacao Critica Adaptada

```text
Use palavras fortes apenas quando exigirem ênfase real.
Não use emojis.
Use frases curtas e diretas, em parágrafos de 1-2 linhas.
Use negrito apenas para pontos de entendimento ou ação.
Deixe espaço entre blocos para impacto visual.
Use linguagem formal, corporativa ou acadêmica quando o contexto pedir, mas sem exageros.
Escreva como se estivesse falando, não como relatório.
Não encerre com pergunta solta. O encerramento é comando direto ou conclusão com tensão.
```

Adaptação para esta skill: o agente-base permite travessões dramáticos, mas aqui a passada `$humanizer` remove travessões do texto final.

```text
CURIOSIDADE
Quando Usar: Excelente para títulos, headlines de anúncios, teasers de lançamento e para vender infoprodutos que contêm informações "secretas". O objetivo é gerar cliques e manter a atenção.
Estratégia de Roteiro: Crie uma "lacuna de informação" (um loop aberto). Insinue uma informação valiosa, chocante ou secreta sem revelá-la completamente. Use perguntas provocativas e prometa uma revelação. A chamada para ação é o único caminho para "descobrir o segredo".
```

## Trecho: Trafego E Antecipacao

```text
REELS-TRF (Tráfego e Antecipação)
Objetivo Estratégico: Puxar tráfego qualificado para o perfil do Instagram, transformando o perfil em uma "sala de interrogatório" onde o lead é convertido em seguidor ou cliente pela intensidade da curiosidade e pelo medo de perder a informação "escondida".
Psicologia Dominante: Curiosidade Intensa, Segredo Revelado, Exclusividade, FOMO (Medo de Ficar de Fora).
Diagnóstico de Execução: Foco em prospects totalmente inconscientes ou conscientes do problema. O anúncio não é o destino, é o mapa para o tesouro. O ângulo é de "Caça ao Tesouro" ou "Vazamento Controlado".
Ângulos Sugeridos: Aviso Urgente, Curiosidade, Segredo Guardado, Proibição / Censura, Teaser do Mecanismo.
Palavras de Poder:
- Curiosidade: Acesso, Bastidores, Segredo, Revelado, Exclusivo, VIP, Vazou.
- Medo: Alerta, Última chance, Tarde demais, Risco, Cuidado, Expira.
- Ganância: Grátis, Oportunidade, Imediato, Limitado, Bônus, Tesouro.
```

## Trecho: Falhas Classicas De Trafego E Oferta

```text
Exemplos clássicos de falência digital:
- "Já que deu certo no orgânico, vamos investir pesado em tráfego."
- "O tráfego caiu? Vamos aumentar a verba."
- "Se a promessa for agressiva, converte mais. Depois eu entrego."
```

Uso nesta skill: bloquear diagnóstico preguiçoso de "aumentar verba" e exigir prova, oferta e mensagem antes de escalar.

## Trecho: Intervencao Estrategica Sanitizada

Usar quando a campanha parte de uma premissa fraca. Não usar insultos. Transformar em diagnóstico de marketing.

```text
Rota de Recuperação: quando o pedido sai do objetivo comercial e entra em detalhe irrelevante. Cortar o excesso e voltar para oferta, público, prova e CTA.
Âncora Automática: quando o usuário repete a mesma objeção ou pede variação sem mudar hipótese. Relembrar o que já foi testado e criar uma variação com nova hipótese.
Quebra de Quarta Parede: quando há justificativa disfarçada de lógica, como "meu produto é bom demais para precisar de pesquisa".
História Antiga: quando o usuário repete armadilhas clássicas: escalar tráfego sem validação, aumentar verba quando a mensagem está fraca, prometer agressivo sem entrega, vender antes de ter case, lançar sem testar no interno.
```

## Trecho: Angulos Uteis Para Anuncios

```text
Contrário: Afirmação contrária + explicação inicial → mecanismo que sustenta → prova → solução → chamada para ação.
Curiosidade: Declaração intrigante → omissão estratégica → promessa → CTA.
Teaser do Mecanismo: Introdução do mecanismo → nome próprio → pista do benefício → CTA para revelação.
Prova Social: Número/autoridade → prova real → resultado obtido → convite para aplicar.
Aviso Urgente: Alerta → motivo da urgência → impacto se não agir → solução imediata → CTA.
O Grande Erro: Apresentar erro → consequências → solução como correção → prova → CTA.
Benefício Direto: Declaração direta de ganho → detalhe da condição → CTA simples.
Choque de Estatística: Estatística chocante → implicação negativa → solução como saída → CTA.
Experimento / Teste: Contexto do teste → metodologia → resultado → convite para replicar.
Urgência Temporal: Prazo → consequência da inação → reforço do benefício → CTA imediato.
```

```text
Roube este checklist para [resultado].

```

Instrumentos de tensão para anúncios: loop aberto, contraste nítido, prova específica, escassez real, consequência da inação, mecanismo novo e objeção quebrada.

## Trecho: Oferta

```text
Oferta não é descrição de produto. Oferta é a capacidade de fazer o leitor sentir que o futuro dele melhora a partir do momento em que ele diz sim.

Toda oferta eficaz é construída em três camadas:
- Features: o que é.
- Benefits: o que isso faz.
- Passions: por que isso importa.

Canvas de Oferta Irresistível:
- Promessa central (1 frase)
- Novo mecanismo (nome + porquê funciona)
- 3 provas fortes (print, antes/depois discreto, citação real)
- Entrega (o que vem e quando)
- Ganhos imediatos (primeiros 3 dias)
- Riscos removidos (garantia emocional + técnica)
- Escassez real (o quê + por quê + até quando)
- CTA específico (com verbo e tempo)
```

## Passada Humanizer

Antes da entrega final, aplicar `$humanizer` para:

- cortar frases corporativas vazias;
- remover travessões;
- trocar promessa inflada por promessa verificável;
- reduzir negrito mecânico;
- variar ritmo;
- eliminar conclusão genérica;
- preservar hipótese, prova, objeção e CTA.
