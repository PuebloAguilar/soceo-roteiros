## Trecho: Personalizacao Do Plano

```text
1. Persona que deseja atingir:
   - Quem é essa pessoa? (Idade, gênero, profissão, etc.)
   - Qual é a sua maior dor/frustração em relação ao meu nicho?
   - Qual é o seu maior desejo/sonho?
   - Que tipo de linguagem ela usa? (Formal, informal, gírias?)

2. Produto/Serviço a ser promovido:
   - O que é?
   - Qual é o benefício principal e transformador dele?
   - Qual o preço/oferta?

3. Objetivo:
   - Vender o produto/serviço diretamente
   - Gerar leads
   - Aumentar o engajamento
   - Construir autoridade/confiança

4. Qual história você quer contar? (O ângulo):
   - Minha própria jornada de transformação
   - A história de um cliente
   - "Quebrando um mito"
   - "Bastidores da criação"
   - Um tutorial rápido que leva à solução completa
```

Uso nesta skill: briefing mínimo para vídeos de YouTube e VSL.

Uso nesta skill: escolher abertura de vídeo.

## Trecho: Regras Para Angulo + Sentimento

```text
O Ângulo é o esqueleto (a estrutura lógica do roteiro) e o Sentimento é a alma (a emoção que dá vida ao roteiro).
A Regra da Aliança: O Ângulo e o Sentimento devem ser compatíveis.
A Regra do Sentimento Dominante: escolha um sentimento principal para guiar o tom.
A Regra da Intensidade: ajuste a força do sentimento ao objetivo do Ângulo.
A Regra da Sinergia Estrutural: aplique o Sentimento com mais força nos pontos-chave da estrutura do Ângulo.
```

## Trecho: Formatacao Critica Adaptada

```text
Use palavras fortes apenas quando exigirem ênfase real.
Não use emojis.
Use frases curtas e diretas, em parágrafos de 1-2 linhas.
Use negrito apenas para pontos de entendimento ou ação.
Deixe espaço entre blocos para impacto visual.
Use linguagem formal, corporativa ou acadêmica quando o contexto pedir, mas sem exageros.
Escreva como se estivesse falando, não como relatório.
Não encerre com pergunta solta. O encerramento é comando direto ou conclusão com tensão.
```

Adaptação para esta skill: o agente-base permite travessões dramáticos, mas aqui a passada `$humanizer` remove travessões do texto final.

## Trecho: Intervencao Estrategica Sanitizada

Usar quando o pedido do usuário revelar desvio, repetição improdutiva, autoengano ou decisão de negócio frágil. Não usar insultos. Transformar a intervenção em diagnóstico objetivo.

```text
Rota de Recuperação: ativar quando o usuário desvia para tema irrelevante ao objetivo. Identificar o desvio, cortar o excesso e retornar ao objetivo.
Âncora Automática: ativar quando o usuário repete uma dúvida já respondida. Relembrar o ponto, aplicar ao contexto atual e avançar.
Quebra de Quarta Parede: ativar quando há autoengano, mentalidade de escassez, crença limitante ou justificativa disfarçada de lógica. Nomear o padrão, mostrar consequência e dar próximo passo.
História Antiga: ativar quando o usuário repete armadilhas clássicas: escalar tráfego sem validação, lançar sem testar, vender mentoria sem case, confiar que audiência resolve oferta, aumentar verba quando a mensagem está fraca, prometer agressivo sem entrega.
```

Aplicação para YouTube: se o roteiro parte de uma tese fraca, abrir um bloco curto de diagnóstico antes de escrever e reposicionar promessa, prova ou CTA.

## Trecho: Angulos Mais Uteis Para YouTube

```text
Contrário: Afirmação contrária + explicação inicial → mecanismo que sustenta → prova → solução → chamada para ação.
Curiosidade: Declaração intrigante → omissão estratégica → promessa → CTA.
Teaser do Mecanismo: Introdução do mecanismo → nome próprio → pista do benefício → CTA para revelação.
Prova Social: Número/autoridade → prova real → resultado obtido → convite para aplicar.
O Grande Erro: Apresentar erro → consequências → solução como correção → prova → CTA.
Como Fazer: Declaração de como fazer → passos rápidos → benefício final → CTA.
Experimento / Teste: Contexto do teste → metodologia → resultado → convite para replicar.
História de Superação: Desafio inicial → luta → descoberta/solução → vitória → CTA.
```

```text
[Número] sinais de que [risco] já começou.
Ninguém te conta isso sobre [tema].
Antes de [ação], faça isso por [tempo curto].
O que [grupo A] faz e [grupo B] ignora.
Roube este checklist para [resultado].
A pergunta que destrava [resultado] agora.
```

Instrumentos de tensão: loop aberto, escassez contextual, contraste nítido, pergunta íntima, contagem regressiva, prova parcial, promessa atrasada e consequência futura.

## Trecho: Oferta

```text
Oferta não é descrição de produto. Oferta é a capacidade de fazer o leitor sentir que o futuro dele melhora a partir do momento em que ele diz sim.

Canvas de Oferta Irresistível:
- Promessa central (1 frase)
- Novo mecanismo (nome + porquê funciona)
- 3 provas fortes (print, antes/depois discreto, citação real)
- Entrega (o que vem e quando)
- Ganhos imediatos (primeiros 3 dias)
- Riscos removidos (garantia emocional + técnica)
- Escassez real (o quê + por quê + até quando)
- CTA específico (com verbo e tempo)
```

## Passada Humanizer

Antes da entrega final, aplicar `$humanizer` para:

- cortar introduções de chatbot;
- remover travessões;
- trocar frases infladas por falas concretas;
- reduzir negrito mecânico;
- variar ritmo;
- eliminar conclusão genérica;
- preservar detalhes específicos, prova e CTA.
